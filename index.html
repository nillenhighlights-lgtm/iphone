Установка на Windows

Подключи VPN на компьютере за 7 минут. Hiddify Next надёжно работает с подписочными ссылками.

1

Скачай Hiddify Next

Открой страницу релизов Hiddify Next на GitHub и скачай Hiddify-Windows-x64-Setup.exe (~70 МБ).

Скачать с GitHub →
💡 Hiddify — open-source клиент. Антивирус может ругаться — это ложная тревога, типичная для VPN-программ. Файл с GitHub релизов безопасен.
2

Установи и запусти Hiddify

Запусти скачанный .exe, жми «Next» во всех окнах. Иконка Hiddify появится в меню «Пуск» и на рабочем столе.

Открой программу. При первом запуске Windows может спросить разрешение на сетевые подключения — нажми «Разрешить».

3

Скопируй свою подписку из бота

В Telegram-боте Балдёж VPN нажми «📡 Моя подписка». Скопируй ссылку:

https://sub.spotsub.ru/sub/abc123def456...

Правый клик по ссылке → «Копировать ссылку», или выдели и нажми Ctrl+C.

💡 Эта ссылка уникальная для тебя. Никому её не давай.
4

Добавь подписку в Hiddify

В Hiddify нажми «+» или раздел «Profiles» → «Add new profile».

Выбери «From Clipboard» — Hiddify вставит ссылку. Или «Add by URL» и Ctrl+V.

Назови профиль (например, «Балдёж VPN») → «Save».

5

Подключись к серверу

На главном экране — список серверов. Выбери любой и нажми Connect.

Когда статус сменится на «Connected» — VPN работает.

💡 В трее Windows появится иконка Hiddify — по правому клику быстрый Connect / Disconnect.
6

Проверь, что всё работает

Открой любой браузер и зайди на 2ip.ru — там покажется новый IP и страна сервера.

Частые вопросы

Антивирус удалил установщик
Это типично для VPN-клиентов: антивирусы ошибочно их считают подозрительными. Файлы с официальной страницы релизов GitHub безопасны. Добавь Hiddify в исключения антивируса и повтори скачивание.
Hiddify не запускается / падает
Установи последнюю версию Visual C++ Redistributable с сайта Microsoft. Также попробуй запуск от имени администратора (ПКМ по ярлыку → «Запуск от имени администратора»).
Подключился, но интернет не работает
Попробуй другой сервер. Также в настройках Hiddify попробуй переключить режим маршрутизации с TUN на System Proxy и обратно.
Можно ли использовать на нескольких устройствах?
Да, на одной подписке работают одновременно 3 устройства — компьютер, телефон, планшет в любой комбинации.
Альтернативы Hiddify на Windows?
Подходят v2rayN, NekoRay, Clash Verge. Все принимают подписочные ссылки — ищи раздел «Подписки» / «Subscriptions».
Hiddify запускается каждый раз при старте Windows
В настройках программы: Settings → General → Auto Start — снять галочку. Или Win+R → shell:startup → удалить ярлык.
